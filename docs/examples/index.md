# Examples

The repository examples show each supported adoption path.

- [`examples/plugin-usage`](https://github.com/mertdogar/super-harness/tree/main/examples/plugin-usage)
  is the shortest host integration.
- [`examples/composed-host`](https://github.com/mertdogar/super-harness/tree/main/examples/composed-host)
  mounts the harness beside a host surface.
- [`examples/auth`](https://github.com/mertdogar/super-harness/tree/main/examples/auth)
  pairs the harness with
  [super-line](https://super-line.dogar.biz) authentication.
- [`examples/web`](https://github.com/mertdogar/super-harness/tree/main/examples/web)
  is a full React application.
- [`examples/plan-board`](https://github.com/mertdogar/super-harness/tree/main/examples/plan-board)
  demonstrates planning, delegation, and approvals.
- [`examples/canvas`](https://github.com/mertdogar/super-harness/tree/main/examples/canvas)
  combines a collaborative CRDT canvas with a clustered harness.
