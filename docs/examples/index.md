# Examples

The repository examples show each supported adoption path.

- `examples/plugin-usage` is the shortest host integration.
- `examples/composed-host` mounts the harness beside a host surface.
- `examples/auth` pairs the harness with super-line authentication.
- `examples/web` is a full React application.
- `examples/plan-board` demonstrates planning, delegation, and approvals.
- `examples/canvas` combines a collaborative CRDT canvas with a clustered harness.
